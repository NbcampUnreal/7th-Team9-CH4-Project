# 7th-Team9-CH4-Project# 🍿 별 팝콘 대탈출

> 큐브 동물들의 좌충우돌 협동 플랫포머 레이스!
> 

---

## 📖 게임 소개

지루한 일상을 보내던 큐브 동물들이 하늘에서 떨어진 신비한 **'별 팝콘'** 의 환상적인 맛에 눈을 뜨고 맙니다!

달콤바삭한 팝콘을 독차지하기 위해 농장 주인이 겹겹이 설치한 아찔한 장애물들을 뚫고 팝콘 저장고를 향해 질주하는 좌충우돌 대탈출 레이스!

**피코파크**와 **폴 가이즈**를 혼합한, 인원 수에 따라 맵과 기믹이 변화하는 **3인칭 협동 플랫포머 게임**입니다.

---

## 🎮 게임 규칙

- **2~3인 팀전**, 최대 **6명** 플레이 가능
- 5개의 맵에서 진행되며, **골인 지점에 먼저 도착한 팀**이 해당 라운드 1점 획득
- 총 5라운드가 끝난 뒤 **팀 점수가 높은 팀**이 최종 우승

## 🎁 아이템

아이템 박스를 먹으면 랜덤으로 아이템을 획득합니다.

| 아이템 | 효과 |
| --- | --- |
| 💨 선풍기 | 상대방을 한 번 날려버림 |
| 🧲 자석 | 상대방을 한 번 끌어당김 |
| 💧 물풍선 | 상대방을 2초간 가둠 |
| 🍠 고구마 | 공중에서 한 번 더 점프 가능 |

## 🗺️ 맵 기믹

- 탑 쌓기 (플레이어가 쌓여 장애물을 넘는 협동 기믹)

- 협동 발판 이동 오브젝트
